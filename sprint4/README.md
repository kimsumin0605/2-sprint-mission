## 스프린트 미션4

### 🛡️ 토큰 기반 인증 & 인가 (with Passport & Prisma)

사용자 인증/인가, 게시글/상품 CRUD, 댓글/좋아요 기능 포함

---

## 🧩 주요 기능

### 🔐 인증/인가

- 회원가입: 이메일/닉네임/비밀번호
- 로그인: Passport를 활용한 JWT 기반 인증
- Access/Refresh Token 발급 및 재발급
- 쿠키에 토큰 저장 및 만료 처리
- 인증이 필요한 요청에 `verifyAccessToken` 미들웨어 적용

### 📦 상품 기능

- 로그인한 유저만 상품 등록 가능
- 본인이 등록한 상품만 수정/삭제 가능
- 상품 좋아요/좋아요 취소
- 좋아요한 상품 목록 조회

### 📝 게시글 기능

- 로그인한 유저만 게시글 등록 가능
- 본인 게시글만 수정/삭제 가능
- 게시글 좋아요/좋아요 취소
- 좋아요한 게시글 목록 조회

### 💬 댓글 기능

- 상품/게시글에 댓글 작성 가능 (로그인 필요)
- 본인 댓글만 수정/삭제 가능

### 👤 유저 기능

- 내 정보 조회 및 수정
- 비밀번호 변경 (기존 비밀번호 확인 포함)
- 내가 등록한 상품 목록 조회
- 비밀번호는 응답에서 절대 노출되지 않음

---

## 🔧 기술 스택

- **Express** – Node.js 웹 프레임워크
- **Prisma** 
- **Passport** – 인증 전략 (JWT 기반)
- **superstruct** – 요청 body 검증
- **JWT** – Access / Refresh Token 처리
- **dotenv** – 환경변수 관리
